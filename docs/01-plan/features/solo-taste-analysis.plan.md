## 1. 배경 및 목적

### 현재 문제
- `/api/match/solo` 가 비교 알고리즘(Jaccard, 코사인)을 단독 유저에게 그대로 적용
- 결과 페이지에서 "취향 싱크로율", "공통 채널" 등 비교 전용 UI가 혼자 분석에도 노출됨
- "나의 취향 분석"이라는 레이블만 바꿨을 뿐, 인사이트 없음

### 기대 효과
- 혼자 분석만으로도 완결된 취향 인사이트 제공
- "내 유튜브 유형: 테크 인사이더형 💻" 같은 공유 욕구를 자극하는 결과
- 궁합 분석 CTA로 자연스러운 전환

## 4. 취향 집중도 지수 (Diversity Index)

```
허핀달 지수(HHI) 역산 방식:
  HHI = Σ(카테고리 비중²)
  HHI = 1.0 → 완전 집중 (한 카테고리만)
  HHI = 0.125 → 완전 균등 (8개 동일)

  다양성 점수 = (1 - HHI) / (1 - 0.125) × 100
  → 0점(완전 집중) ~ 100점(완전 다양)
```

집중형(0~35): "취향이 뚜렷한 전문가 타입"
균형형(36~65): "여러 장르를 골고루 즐기는 타입"
다양형(66~100): "모든 것에 열려있는 취향 콜렉터"
